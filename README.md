# Praktikum-Perancangan-Jaringan — Network Design Practicum

> 📚 Practicum from **Internet Engineering Technology**, Universitas Gadjah Mada  
> 🛠️ Tools: Cisco Packet Tracer


## 📖 About This Repository

This repository contains network design practicum exercises from UGM, implemented using **Cisco Packet Tracer**. Each session was based on real-world case studies provided by the lab instructor, focusing on designing, configuring, and troubleshooting network topologies from scratch.


## 🧠 Topics & Case Studies Covered

**Network Fundamentals**
- OSI and TCP/IP model implementation
- IP addressing and subnetting (IPv4 & IPv6)
- Network topologies: star, ring, bus, mesh, hybrid

**Routing & Switching**
- Static and dynamic routing (RIP, OSPF, EIGRP)
- VLAN configuration and inter-VLAN routing
- Spanning Tree Protocol (STP)
- Router and switch configuration via CLI

**Network Protocols**
- DHCP server configuration
- DNS resolution setup
- NAT/PAT (Network Address Translation)
- ACL (Access Control Lists) for traffic filtering
- FTP, HTTP, and email server simulation

**WAN & Advanced Concepts**
- Wide Area Network (WAN) topology design
- VPN basics
- Network redundancy and failover planning

**Security Configurations**
- Implementing ACLs as basic firewall rules
- Port security on switches
- SSH configuration for secure remote access


## 🛠️ Tools & Environment

| Tool | Usage |

## 🚀 How to Open

1. Download and install [Cisco Packet Tracer](https://www.netacad.com/cisco-packet-tracer) (free via NetAcad account)
2. Open any `.pkt` file from this repository
3. Explore the topology, configurations, and simulation mode


## 💡 Key Learning Outcomes

- Able to design complete network topologies from case study requirements
- Experienced with Cisco CLI for router and switch configuration
- Understands routing protocol behavior and convergence
- Applies network security basics (ACL, port security, SSH) to real designs
- Builds strong foundation for network security and infrastructure defense
